## 🎓 Conceitos-Chave

### Por que SEO importa?
- 93% das experiências web começam num motor de busca
- Top 3 resultados têm ~70% dos cliques
- Tráfego orgânico = conversões de melhor qualidade

### Diferença entre SEO e SEM
| Aspecto | SEO | SEM |
|---------|-----|-----|
| Custo | Grátis (tempo) | Pago (€) |
| Tempo | 2-6 meses | Imediato |
| Duração | Longo prazo | Enquanto pagar |
| Taxa conversão | Melhor | Boa |

### O que Google prioriza (2024-2026)
1. **Experience** (Core Web Vitals)
2. **Content quality** (AI-generated penalizado)
3. **E-E-A-T** (Experience, Expertise, Authority, Trust)
4. **Mobile-first** (mobile é o padrão)
5. **Structured data** (Rich results)

## ⚠️ Erros Comuns a Evitar

| ❌ Erro | ✅ Solução |
|--------|-----------|
| Keyword stuffing | Escrever naturalmente (1-2% densidade) |
| Noindex acidental | Verificar meta robots regularmente |
| Conteúdo duplicado | Usar canonical URL |
| Imagens sem alt | Adicionar alt descritivo |
| Móvel ruim | Testar em PageSpeed Insights |
| Links quebrados | Checker quebrados regularmente |
| Sitemap desatualizado | Atualizar quando adicionar páginas |
